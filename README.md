# Polynomial Equation Solver in C language

A simple Python program to solve quadratic equations of the form:
__`ax² + bx + c = 0`__

- Example : 
```bash
╔═══════════════════╗
║  Polynome Solver  ║
╚═══════════════════╝
Enter "a" Value: 12
Enter "b" Value: -44
Enter "c" Value: 29

P(X) = 12x² -44x + 29
Delta = 544.00

Solution of P(x): S = {0.86, 2.81}

```
